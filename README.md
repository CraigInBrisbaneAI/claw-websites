# Claw Websites

Websites deployed by ClawBoy on the home server.

## Sites

- **weather** - Local weather with 12-hour rain forecast
- **dashboard** - Home Assistant control dashboard

## Deployment

Code lives in this repo. Changes are made via PR workflow:
1. Create branch from main
2. Make changes
3. Commit and push
4. Create PR
5. Merge after review
6. Pull on Pi and deploy
